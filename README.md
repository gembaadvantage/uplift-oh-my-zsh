# uplift-oh-my-zsh

Get all the [`uplift`](https://github.com/gembaadvantage/uplift) autocompletion goodness with this oh-my-zsh plugin.

## Quick Start

Clone to the oh-my-zsh plugins folder:

```sh
git clone https://github.com/gembaadvantage/uplift-oh-my-zsh ~/.oh-my-zsh/plugins/uplift
```

Enable it within your `~/.zshrc` file:

```zsh
plugins=(... uplift)
```

Fin!
